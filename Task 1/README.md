# Requirements

Before running the project, ensure the following dependencies and environment requirements are satisfied.

## Software Requirements
- **Python:** Python 3.8 or higher  
  - Required for support of `os.add_dll_directory()`
- **C++ Compiler:** `g++` (GNU Compiler Collection) with **C++17** support

## File Structure
The following files **must be located in the same directory**:
game_nnue_v2_5.py
engine_nnue_v2_5.cpp
nnue_weights_v2_5.h


## To Run

1. Install the required dependencies:

```bash
pip install -r requirements.txt

python game_nnue_v2_5.py
